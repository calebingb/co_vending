# co_vending
FiveM script made to get the vending machines to work!

## How To Install
```
cd resources
git clone https://github.com/badgerfan3/co_vending
```

## How to Run
```
Add to server.cfg:

start co_vending
```

## All code to change:

Change "bt-target" in the fxmanifest to the location of pf-target in the resource files
